## One-liner

Ansible repo that provisions a single-DO-Droplet hosting environment for BitSalt's portfolio of WordPress / Laravel / Node.js / FastAPI sites behind a shared Traefik proxy.

## Stack

- **Language:** Ansible (YAML), Bash (bootstrap.sh).
- **Framework:** Ansible 2.15+, `community.docker` 5.2.0.
- **Tests:** `ansible-lint` + `yamllint` (no functional tests).
- **Deploy:** This repo *is* the deploy mechanism (`ansible-playbook site.yml --ask-vault-pass`). CI runs lint only.
